Technology Best Practices

Environment Usage Best Practices

1
Environment Selection

Always select the appropriate environment for your development phase. Use DEV environments for development, SC for security testing, and TS for pre-production validation.

Tip: Match your environment to your current development phase to ensure proper resource allocation and security compliance.
2
Resource Management

Practice efficient resource management by cleaning up unused resources and optimizing workloads. Release compute resources when not actively developing.

Tip: Set up automated resource cleanup tasks to prevent resource wastage and reduce costs.
